Paper: "Lunar Base Operations Using Decentralized Network of Robots Controlled Using Physics Informed Artificial Neural Networks"

Contribution: Contributed to the core formulation of the neural network used for task allocation in a multi-robot system. Designed a scalable input encoding method using fixed-size padding, allowing the ANN to handle varying numbers of robots. This involved normalizing inputs by placing active robot positions followed by padded placeholders (mapped to zero-task outputs), ensuring training consistency. The resulting ANN architecture enabled robust task delegation in dynamic lunar environments, inspired by similar techniques I developed during prior work on multi-robot genetic algorithms. Inputs for the forecasting ANN were not part of my direct implementation, though I shared some initial feedback.
