Conferences & Other Contributions

🚀 ISSC Conference on Multi-Robot Systems 2024

Paper: "Task Planning for Lunar Surface Construction using Multi-Robot Swarms"

Summary: The paper presents a decentralized task planning algorithm for a swarm of autonomous robots tasked with constructing semi-permanent structures on the lunar surface. It addresses challenges like robot blockage, task sequencing, and failure recovery using a genetic algorithm-based optimizer. The proposed framework ensures optimal task distribution and coordination, with simulation results validating robustness under off-nominal conditions.

📊 NCMDAO 2023 Presentation

Paper: "Quantifying Player Performance and Optimizing Team Selection in Fantasy T20 Cricket"

Summary: This work proposes a scoring model for fantasy cricket that incorporates both historical and recent player performance, pitch conditions, and opposition strength. Using a weighted exponential average and a constraint-enhanced genetic algorithm, the system optimizes team selection in real time. The method achieves higher prediction accuracy compared to previous static approaches, especially for dynamic match scenarios in the IPL.

Non-Author Contributions

Paper: "Lunar Base Operations Using Decentralized Network of Robots Controlled Using Physics Informed Artificial Neural Networks"

Contribution: Contributed to the neural network formulation for multi-robot task allocation by designing a scalable input strategy using fixed-size padding. Enabled consistent training across varying team sizes and improved real-time task assignment efficiency.

Paper: "Design and Development of SMART Architecture for Lunar Bases"

Contribution: Formulated a genetic algorithm-based optimization strategy for sensor placement in 2D lunar base layouts. Improved scalability and convergence by proposing a binary chromosome structure and custom fitness function addressing redundancy and interference.
